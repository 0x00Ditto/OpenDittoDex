# OpenDittoDex
DittoDex is a OpenSource Project which aims to create a pokedex resembling that of Pokemon Kanto
<hr />

## Feature area : 
- [x] Search by ID 
- [x] Search by Name
- [x] Sprite update
- [x] Pokemon Type
- [x] Stats Table
- [x] Pad directional
- [ ] Search with Button ( 0 to 9 )
- [ ] Animation (Pokeball, fake time search, alarm blue,red,yellow,green)
- [ ] Random Search
- [ ] Pokemon voice
- [ ] ??? Easter Egg ???
